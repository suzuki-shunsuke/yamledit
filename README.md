# yamledit

yamledit is a CLI to edit YAML files based on declarative migration files while keeping YAML comments and indentation. It's useful to automate the migration of YAML files. It allows you to distribute migration files so that anyone can use them. Users can leverage shared migration files instead of writing migration rules from scratch.

## Install

```sh
go install github.com/suzuki-shunsuke/yamledit/cmd/yamledit@latest
```
